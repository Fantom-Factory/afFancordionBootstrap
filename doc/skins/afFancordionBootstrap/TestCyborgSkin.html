Rendering Example

Cyborg Skin

This specification shows the different outputs generated by the Cyborg Fancordion Skin.

When a command is successful it is shown in Green.

When a command fails it is shown in Green Red.

Sometimes the unexpected happens and a command throws an Exception Err: Boom! In yer face!

Example: Pre Text

Pre-formatted text is useful for showing:

System messages and verbatim text

It may also be used for verifications:

Green for success!
