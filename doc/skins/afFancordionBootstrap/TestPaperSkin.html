Rendering Example

Paper Skin

This specification shows the different outputs generated by the Paper Fancordion Skin.

When a command is successful it is shown in Green.

When a command fails it is shown in Green Red.

Sometimes the unexpected happens and a command throws an Exception Err: Boom! In yer face!

Example: Tables

Fancordion allows a simple table markup to be used in the specifications. Tables may be used to run the same command over multiple rows of data:

First NameLast NameFull Name
JoannaStormJoanna Storm
RachelAshleyRachel Ashley
AmberLynnAmber Lynn
BarbaraDareBarbara Dare
BrittMorganBritt Morgan Nina Hartley
